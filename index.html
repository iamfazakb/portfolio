Forms

Forms are used to receive user input. The method attribute can be either get or post. The get method is ideal for:

  • short forms
  • when you are just retrieving data from the web server
As a rule of thumb you should use the post method if your form:
  • allows users to upload a file
  • is very long
  • contains sensitive data
  • adds information to, or deletes information from, a database
